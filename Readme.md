<h1>Introduction</h1>
This is a Hospital Management System built in C++ that provides a comprehensive solution for managing hospital operations. The system allows administrators, doctors, patients, and reception staff to perform various tasks efficiently, including:

Patient Management: Register, view, and manage patient records.

Doctor Management: Add and view doctors and their specializations.

Appointment Scheduling: Schedule and view appointments between patients and doctors.

Room Management: Assign patients to rooms and manage ward availability.

Billing System: Generate and view patient bills including doctor, room, and medication charges.

The project was developed by our team as part of First Semester Project for BSIT
